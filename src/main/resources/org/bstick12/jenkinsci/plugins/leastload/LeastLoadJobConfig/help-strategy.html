Specify a strategy the plugin should use when selecting an executor for this job only.
  • Use Global - Use the strategy selected in your Jenkins global configuration.
  • Prefer Idle - Prefer machines that are totally idle, or if none, the machine with the most idle executors available.
  • Round Robin - Prefer machines that have gone the longest without being assigned a job, ensuring that all machines are assigned jobs when possible.
  • Disabled - Disable the plugin and fall back to the default Jenkins load balancer.
